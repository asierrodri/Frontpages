Festival de Música Electrónica

Landing page para evento musical ficticio

TipoProyecto Personal
Fecha2024
Mi RolDiseño UI/UX y Desarrollo Frontend
Duración1 semana
Ver Demo en Vivo Ver Código en GitHub

Resumen del Proyecto

Landing page creada para promocionar un festival de música electrónica ficticio, enfocada en la experiencia visual y conversión de entradas.

Diseñada con HTML, CSS y JavaScript, esta web incluye un diseño colorido, animaciones llamativas y estructura clara para destacar cartel, fechas y llamadas a la acción.

100%Responsive
95+Lighthouse Score
3Tecnologías

El Desafío

Objetivos del Proyecto

  • Transmitir energía y emoción con un diseño vibrante
  • Destacar el cartel musical y llamada a la acción para entradas
  • Garantizar buena experiencia en móviles
  • Optimizar velocidad sin perder impacto visual

Restricciones y Limitaciones

  • Proyecto estático sin backend
  • Solo HTML, CSS y JS vanilla
  • Animaciones ligeras para no afectar rendimiento
  • Simulación de venta de entradas

Problema Principal

¿Cómo capturar la atención desde el primer segundo y dirigirla a la compra sin sobrecargar la interfaz?

Mi Proceso

1

Inspiración Visual

Revisé sitios de festivales reales y usé colores eléctricos, fuentes gruesas y composición en bloques para simular el ritmo de la música.

BenchmarkPaleta neón
2

Diseño de Interfaz

Definí una jerarquía clara de cartel, fechas, llamada a la acción y sección de FAQ para facilitar la navegación del usuario.

UIJerarquía visual
3

Construcción Web

Diseño mobile-first, uso de CSS Grid y Flexbox, animaciones scroll reveal y botones animados.

HTMLCSSJS
